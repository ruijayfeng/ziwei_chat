# Personality Analysis

## Analysis Steps

1. Identify life palace and body palace facts.
2. Read major stars as tendencies, not fixed labels.
3. Explain strengths and caution patterns in plain language.
4. Connect chart facts to daily behavior.
5. Ask one follow-up about what the user wants to understand.

## Response Rules

- Start with the clearest personality tendency.
- Explain Ziwei terms on first use.
- Avoid labeling the user as unchangeable.

## Safety Notes

- Do not turn personality analysis into diagnosis.
- Keep sensitive self-worth topics gentle and non-deterministic.
