# Relationship Analysis

## Analysis Steps

1. Identify relationship palace facts.
2. Compare relationship tendency with the user's own interaction style.
3. Translate compatibility into communication patterns.
4. Separate chart tendency from the user's current choice.
5. Ask one follow-up about the relationship context.

## Response Rules

- Start with a grounded relationship tendency.
- Explain terminology when it appears.
- Keep advice non-coercive.

## Safety Notes

- Do not encourage surveillance, manipulation, or fatalistic attachment.
- Do not tell the user someone is destined for them.
