赛道全景：AI 命理正从"玩具"走向"工具"

2024–2026 年是 AI 命理赛道的爆发窗口。全球 AI 占星 App 市场在 2024 年达到 8.51 亿美元，预计 2031 年将增长至 39.72 亿美元，年复合增长率 26.4%。中国市场更是呈现"井喷"态势——仅参天 AI 一家就在 2025 年宣布全球用户突破 500 万、日均查询量超 10 万次。头部产品的付费转化率达到 18%，远超知识付费行业平均水平。

但这波浪潮有一个根本性的结构问题：90% 以上的所谓"AI 命理产品"本质上是 Prompt Wrapper——在通用大模型（DeepSeek、GPT-4、Claude）外面包了一层命理提示词。这导致了两个致命缺陷：(1) 干支历法数学幻觉——LLM 排盘错误率极高；(2) 解读缺乏确定性和可复现性——同一个命盘、同一个问题，两次回答可能完全矛盾。

产品经理洞察：市场正在经历从"能用 AI 算命吗？"到"哪个 AI 算得准？"的认知升级。用户已经开始意识到通用大模型做命理的不可靠性。这意味着确定性排盘引擎 + 结构化分析流程将成为下一阶段的竞争壁垒。Ziwei Chat 的架构决策恰好踩在了这个趋势的前沿。

04

SWOT 分析：Ziwei Chat 的真实处境

Strengths · 优势

  • 架构护城河最深：9 步 Agent 管道（意图路由→规划→工具→知识→审查）是真正的垂直 Agent，非 Prompt Wrapper
  • 准确性链完整：出生数据→确定性排盘→命盘事实→分析工作流→审查→回答，每步可审计
  • 安全设计行业唯一：三级安全边界 + 内置审查器，在命理赛道是独一无二的责任设计
  • 证据透明：Evidence Companion 让用户看到"系统用了什么数据"，建立信任
  • 开源 + 零门槛：无需注册、无需付费、无需嵌入服务即可运行，降低获客摩擦
  • Vercel 部署友好：一键部署是开源项目的分发杠杆
  • 产品语调差异化："温暖朋友 + 领域能力"定位独特，不神秘不权威

Weaknesses · 劣势

  • 功能覆盖面窄：仅紫微斗数单一术数，无法满足用户"八字+紫微+风水"一站式需求
  • 内容深度不足：14 星曜×12 宫位矩阵未完整覆盖，四化×宫位矩阵缺失
  • 无商业化路径：纯开源、零注册、零付费，缺乏可持续的商业模式
  • 仅中文：无法触达海外华人或对东方命理好奇的国际用户
  • 无移动端：仅 Web 端，在移动优先的中国市场是明显短板
  • 社区/生态为零：无开发者社区、无插件生态、无第三方内容贡献
  • 品牌认知度为零：无市场推广、无 SEO、无社交媒体存在

Opportunities · 机会

  • 确定性 AI 命理是蓝海：90% 竞品是 Prompt Wrapper，真正确定性排盘+Agent 管道是稀缺能力
  • 开源社区杠杆：iztro 生态 + Vercel 部署 = 技术圈传播的天然路径
  • AI 算命信任危机：用户正在觉醒——"DeepSeek 排盘老出错"是 Ziwei Chat 的天然获客场景
  • 知识贡献 UGC：命理爱好者/从业者可以贡献内容，形成护城河
  • 企业/B2B 场景：命理 SaaS API 供其他产品集成
  • 全球化潜力：紫微斗数在东南亚（台湾、香港、马来西亚、新加坡）有深厚文化基础
  • 多术数扩展：架构可扩展至八字、风水等其他术数

Threats · 威胁

  • 通用大模型的快速进化：如果 GPT-5/Claude 4 内置了干支历法函数调用能力，Prompt Wrapper 也能变准
  • 天府 Agent 的先发优势：250+ 工具、多流派、商业化成熟，可能形成网络效应
  • 参天 AI 的规模碾压：500 万用户 + 12 种语言 + iOS App + 资本加持
  • 政策风险：中国对"算命"类内容的监管可能收紧
  • iztro 依赖风险：核心排盘引擎依赖第三方开源项目，项目停更或方向变更会影响产品
  • 开源被"借鉴"：竞品可以直接 fork 你的 Agent 管道和知识内容

战略定位：Ziwei Chat 的"唯一真解"

定位宣言

Ziwei Chat 不是"又一个 AI 算命工具"。它是全球唯一一个开源、确定性、可审计的紫微斗数垂直 Agent。它的竞品不是天府 Agent 或参天 AI，而是"用 DeepSeek 算命的用户习惯"——它要做的不是抢存量用户，而是定义"专业 AI 命理"这个新品类。

Ziwei Chat 占据的是"专业深度 × 准确可靠"象限——这是当前市场最大的空白地带。天府 Agent 也在这个方向，但它偏"从业者工具"，而 Ziwei Chat 偏"消费者产品"。两者的关键差异在于：天府 Agent 追求"功能多"（250+ 工具、三流派），Ziwei Chat 追求"体验好"（温暖语调、证据透明、零门槛）。

核心差异化策略

🎯 策略一：定义品类

不跟参天 AI 比用户量，不跟天府 Agent 比工具数。而是定义一个新品类——"可审计的 AI 命理"。让用户形成认知："真正准的 AI 算命，是能看到每一步推理过程的。"

🔓 策略二：开源即获客

开源不是情怀，是获客策略。每一个在 GitHub 上 star 项目的开发者，都是一个潜在的高质量传播节点。Vercel 一键部署让技术用户可以"自部署一个自己的算命 Agent"。

🛡️ 策略三：安全即信任

在"AI 算命乱象"中，三级安全边界 + 审查器是 Ziwei Chat 最强的品牌资产。当竞品在追求"说得更玄"时，Ziwei Chat 在追求"说得更负责任"——这是长期信任的根基。

📚 策略四：内容社区化

知识内容（content/ 目录）可以成为 UGC 入口。命理爱好者/从业者可以贡献 curated 知识块。高质量内容贡献者 = 高质量传播者 = 自然增长飞轮。

产品建议：从现在到 V2 的关键动作

短期（V1 发布前 · 1–2 个月）

补齐内容缺口

14 星曜×12 宫位矩阵和四化×宫位矩阵是回答质量的根基。没有这些，Agent 在具体分析时会"言之无物"。这是最高优先级的内容工作。

打磨证据体验

Evidence Companion 是 Ziwei Chat 的"签名特征"。让证据抽屉不仅展示工具调用，还要展示"为什么选择这个工具"和"知识来源的可信度评分"。

差异化首页

首页要讲清楚"为什么 Ziwei Chat 比通用 AI 算得准"。用对比示例（同一命盘、同一问题，Ziwei Chat vs 通用大模型的回答差异）建立信任。

中期（V1→V2 · 3–6 个月）

移动端 PWA

不需要原生 App，但必须做好 PWA（离线可用、添加到主屏幕、推送通知）。中国市场的移动 Web 体验直接决定留存率。

知识贡献机制

开放 content/knowledge/ 的贡献流程（GitHub PR + 审核）。让命理爱好者可以提交 curated 知识块。贡献者 = 传播者。

SEO + 内容营销

围绕"AI 算命哪个准""紫微斗数排盘工具""DeepSeek 算命准吗"等关键词做 SEO 内容。这是零成本获客的核心渠道。

多语言 MVP（繁体中文）

优先支持繁体中文，直接触达台湾、香港、马来西亚、新加坡市场。紫微斗数在这些地区的文化基础远深于大陆。

长期（V2+ · 6–12 个月）

商业化探索

开源核心 + 托管服务付费（类似 Supabase/Next.js 模式）。自部署免费，官方托管按月订阅。企业 API 供第三方集成。

多术数扩展

架构已支持 Skills 插件化。可逐步添加八字、风水、奇门遁甲等术数的分析工作流，但每个术数必须遵循相同的"确定性引擎+审查"原则。

社区生态

建立 Discord/微信群社区，培育开发者 + 命理爱好者双社区。开发者贡献代码和工具，命理爱好者贡献知识和反馈。

风险提醒：Ziwei Chat 当前最大的风险不是竞品，而是"完成度"。一个功能 80% 完成的开源项目和功能 100% 完成的商业产品，用户会选择后者。V1 发布前必须确保：五大话题可用、证据抽屉完整、回答质量稳定。宁可功能少，不能体验差。

最终判断：Ziwei Chat 能赢吗？

从 10 年产品经理的视角来看，Ziwei Chat 的产品基因是优秀的——它踩准了三个关键趋势：

  1. 01
    从"Prompt Wrapper"到"确定性 Agent"的范式转换 用户已经受够了 AI 算命胡说八道。确定性排盘 + 结构化分析是下一阶段的入场券。
  2. 02
    从"黑箱算命"到"透明推理"的信任升级 证据抽屉 + 审查器 = 可审计的 AI。这在命理赛道是独一无二的信任资产。
  3. 03
    从"封闭 SaaS"到"开放生态"的分发革命 开源 + Vercel 部署是开发者传播的天然杠杆。参天 AI 花几百万买量，Ziwei Chat 靠 GitHub star 获客。

但基因好不等于能赢。Ziwei Chat 面临三个生死问题：

  1. ⚠
    完成度 vs 时间窗口 天府 Agent 和参天 AI 不会停下来等你。V1 必须在 2 个月内达到"可自豪地推荐给朋友"的质量水平。
  2. ⚠
    获客：谁来用？怎么知道？ 好产品不等于有人用。需要明确的 GTM 策略——SEO、技术社区、命理社区、社交媒体——至少选一个打透。
  3. ⚠
    可持续性：谁来维护？怎么赚钱？ 纯开源 + 零商业化的模式在长期不可持续。需要在 V2 阶段探索托管服务或 API 付费模式。

最终结论：Ziwei Chat 是目前市场上架构最完整、理念最先进的 AI 紫微斗数产品。它定义了一个正确的方向——确定性排盘 + Agent 管道 + 可审计推理。但正确的方向不等于商业成功。接下来的 6 个月，团队需要在"产品完成度"和"用户获取"两个维度上全力冲刺。如果能在 V2 之前积累 1000 个活跃用户和 500 个 GitHub star，Ziwei Chat 将在这个 400 亿美元的赛道中占据一个难以撼动的位置。
